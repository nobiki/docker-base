# react-base

#### Products used

* Infrastracture: `Docker`
* Web: `Nginx`
* Server-side: `golang(gin)`
* Front-end: `react`

## node

```
// First time only
npm i --save-dev create-react-app
npx create-react-app app
```

```
npm install --prefix app
npm run --prefix app build
```

## go

```
// First time only
go mod init api
```

```
go mod tidy
```
